<InstallManual/>


### Download

The macOS versions of **xPack Docus Mock**
are packed as `.tar.gz` archives.
Download the latest version named like:

<ul>
<li><code>xpack-docus-mock-{ props.version }-{ props.xpack_subversion }-darwin-x64.tar.gz</code></li>
<li><code>xpack-docus-mock-{ props.version }-{ props.xpack_subversion }-darwin-arm64.tar.gz</code></li>
</ul>

### Unpack

<p>To manually install the xPack Docus Mock,
unpack the archive and move it to <code>~/.local/xPacks/docus-mock/xpack-docus-mock-{ props.version }-{ props.xpack_subversion }</code>:</p>

```bash
mkdir -p ~/.local/xPacks/docus-mock
cd ~/.local/xPacks/docus-mock

tar xvf ~/Downloads/xpack-docus-mock-{ props.version }-{ props.xpack_subversion }-darwin-x64.tar.gz
chmod -R -w xpack-docus-mock-{ props.version }-{ props.xpack_subversion }
```

:::info
For manual installs, the recommended
install location is different from the xpm install folders.
:::

The result is a structure like:

```bash
$ tree -L 2 /Users/ilg/.local/xPacks/docus-mock/xpack-docus-mock-{ props.version }-{ props.xpack_subversion }
/Users/ilg/.local/xPacks/docus-mock/xpack-docus-mock-{ props.version }-{ props.xpack_subversion }/
├── README.md
├── bin
│   └── docus-mock
└── distro-info
    ├── CHANGELOG.md
    ├── licenses
    ├── patches
    └── scripts

5 directories, 3 files
```
