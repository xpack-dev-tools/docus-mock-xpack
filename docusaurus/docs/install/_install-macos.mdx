<InstallManual/>


### Download

The macOS versions of **xPack Docus Mock**
are packed as `.tar.gz` archives.
Download the latest version named like:

- `xpack-docus-mock-{{ page.version }}-{{ page.xpack-subversion }}-darwin-x64.tar.gz`
- `xpack-docus-mock-{{ page.version }}-{{ page.xpack-subversion }}-darwin-arm64.tar.gz`

### Unpack

To manually install the xPack Docus Mock,
unpack the archive and move it to
`~/.local/xPacks/docus-mock/xpack-docus-mock-{{ page.version }}-{{ page.xpack-subversion }}`:

```sh
mkdir -p ~/.local/xPacks/docus-mock
cd ~/.local/xPacks/docus-mock

tar xvf ~/Downloads/xpack-docus-mock-{{ page.version }}-{{ page.xpack-subversion }}-darwin-x64.tar.gz
chmod -R -w xpack-docus-mock-{{ page.version }}-{{ page.xpack-subversion }}
```

{% include note.html content="For manual installs, the recommended
install location is different from the xpm install folders." %}

The result is a structure like:

```console
$ tree -L 2 /Users/ilg/.local/xPacks/docus-mock/xpack-docus-mock-{{ page.version }}-{{ page.xpack-subversion }}
/Users/ilg/.local/xPacks/docus-mock/xpack-docus-mock-{{ page.version }}-{{ page.xpack-subversion }}/
├── README.md
├── bin
│   └── docus-mock
└── distro-info
    ├── CHANGELOG.md
    ├── licenses
    ├── patches
    └── scripts

5 directories, 3 files
```
