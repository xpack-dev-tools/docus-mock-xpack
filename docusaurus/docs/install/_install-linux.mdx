### Download

The GNU/Linux versions of **xPack Docus Mock**
are packed as `.tar.gz` archives.
Download the latest version named like:

<ul>
<li><code>xpack-docus-mock-{ props.version }-{ props.xpack_subversion }-linux-x64.tar.gz</code></li>
<li><code>xpack-docus-mock-{ props.version }-{ props.xpack_subversion }-linux-arm.tar.gz</code></li>
<li><code>xpack-docus-mock-{ props.version }-{ props.xpack_subversion }-linux-arm64.tar.gz</code></li>
</ul>

As the name implies, these are GNU/Linux `tar.gz` archives; they were build on
Ubuntu, but can be executed on most recent GNU/Linux distributions.

### Unpack

<p>To manually install the xPack Docus Mock,
unpack the archive and move it
to <code>~/.local/xPacks/docus-mock/xpack-docus-mock-{ props.version }-{ props.xpack_subversion }</code>:</p>

<pre><code className="language-console">{
`mkdir -p ~/.local/xPacks/docus-mock
cd ~/.local/xPacks/docus-mock

tar xvf ~/Downloads/xpack-docus-mock-${ props.version }-${ props.xpack_subversion }-linux-x64.tar.gz
chmod -R -w xpack-docus-mock-${ props.version }-${ props.xpack_subversion }
`}</code></pre>

:::info
For manual installs, the recommended
install location is slightly different from the xpm install folders,
which use the scope (like `@xpack-dev-tools`) to group different tools,
and `.content` to store the unpacked archive."
:::

<pre><code className="language-console">{
`$ tree -L 2 /home/ilg/.local/xPacks/docus-mock/xpack-docus-mock-${ props.version }-${ props.xpack_subversion }
/home/ilg/.local/xPacks/docus-mock/xpack-docus-mock-${ props.version }-${ props.xpack_subversion }/
├── bin
│   └── docus-mock
├── distro-info
│   ├── CHANGELOG.md
│   ├── licenses
│   ├── patches
│   └── scripts
└── README.md

5 directories, 3 files
`}</code></pre>
